Sequences documented in parens are implicit ones from parser.c, which move
between states.

1 = VT100
 
 C0 controls
  
1   0x00             = NUL
1   0x07             = BEL
1   0x08             = BS
1   0x09             = HT
1   0x0A             = LF
1   0x0B             = VT
1   0x0C             = FF
1   0x0D             = CR
1   0x0E             = LS1
1   0x0F             = LS0
   (0x18             = CAN)
   (0x1A             = SUB)
   (0x1B             = ESC)

1   0x7f             = DEL (ignored)
  
 C1 controls
  
1   0x84             = IND
1   0x85             = NEL
1   0x88             = HTS
1   0x8D             = RI
   (0x90             = DCS)
   (0x9B             = CSI)
   (0x9D             = OSC)
  
 Escape sequences
  - excluding sequences that are C1 aliases
  
1   ESC ()           = SCS, select character set (G0, G1)
    ESC *+           = SCS, select character set (G2, G3)
1   ESC 7            = DECSC - save cursor
1   ESC 8            = DECRC - restore cursor
1   ESC # 8          = DECALN
1   ESC =            = DECKPAM, keypad application mode
1   ESC >            = DECKPNM, keypad numeric mode
   (ESC P            = DCS)
   (ESC [            = CSI)
   (ESC ]            = OSC)
    ESC n            = LS2
    ESC o            = LS3

 Graphic Renditions
  
1   SGR 0            = Reset
1   SGR 1            = Bold on
    SGR 3            = Italic on

1   SGR 7            = Reverse on
    SGR 9            = Strikethrough on
    SGR 10-19        = Select font
    SGR 21           = Underline double
    SGR 22           = Bold off
    SGR 23           = Italic off
    SGR 24           = Underline off
    SGR 25           = Blink off
    SGR 27           = Reverse off
    SGR 29           = Strikethrough off
    SGR 30-37        = Foreground ANSI
    SGR 38           = Foreground alternative palette
    SGR 39           = Foreground default
    SGR 40-47        = Background ANSI
    SGR 48           = Background alternative palette
    SGR 49           = Background default
    SGR 90-97        = Foreground ANSI high-intensity
    SGR 100-107      = Background ANSI high-intensity
 
The state storage used by CSI s, ESC 7 and DECSM 1048/1049 is shared.
